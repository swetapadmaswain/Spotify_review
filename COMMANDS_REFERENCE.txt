DATABASE OPERATIONS
================================================================================

Connect to Database:
  psql $DATABASE_URL

  psql $DATABASE_URL -c "SELECT COUNT(*) FROM pattern_insights"

  psql $DATABASE_URL -c "SELECT COUNT(*) FROM user_segments"

List All Tables:
  psql $DATABASE_URL -c "\dt"

View Table Schema:
  psql $DATABASE_URL -c "\d table_name"

Query Raw Data:
  psql $DATABASE_URL -c "SELECT * FROM raw_reviews LIMIT 10"

Clear Insights (WARNING: Destructive):

================================================================================
TROUBLESHOOTING
================================================================================

Fix Python Module Not Found:
  set PYTHONPATH=%CD%
  python scripts/start_backend.py

Fix npm Module Not Found:
  cd dashboard && npm install

Fix Database Connection:
  python -c "from app.database.connection import init_db; init_db()"

Clear npm Cache:
  npm cache clean --force

Reinstall Dashboard Dependencies:
  cd dashboard && rm -r node_modules && npm install

Restart Everything:
  # Kill all processes
  pkill -f "python scripts/start_backend.py"
  pkill -f "npm run dev"
  
  # Start fresh
  python scripts/start_backend.py
  cd dashboard && npm run dev

Fix CORS Issues:
  # Already configured in app/api/server.py
  # If still issues, verify backend is running on http://localhost:8000

Fix "Port Already in Use":
  # Port 8000 in use
  lsof -i :8000
  kill -9 <PID>
  
  # Port 5173 in use (dashboard)
  lsof -i :5173
  kill -9 <PID>
